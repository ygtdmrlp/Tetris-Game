About Android Tetris

This is a modern implementation of the classic Tetris game, designed with an Android-like interface. The game features a clean, material design inspired look and responsive controls.

Features

  • Modern, Android-style interface
  • Next piece preview
  • Score tracking
  • Responsive design
  • Keyboard controls

Developer Information

Created in 2025

This game is open source and available under the MIT License. Feel free to contribute to the project or use the code in your own projects.

How to Play

  • Use left and right arrow keys to move pieces
  • Up arrow key to rotate pieces
  • Down arrow key to make pieces fall faster
  • Clear lines to score points
  • Game ends when pieces stack to the top
